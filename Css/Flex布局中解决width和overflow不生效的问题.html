# 4.Flex 布局中解决 width 和 overflow 不生效的问题

# 一、子元素使用 flex:none 解决

<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>flex</title>

    <style>
        .box {
            //父元素使用flex
            display: flex;
            width: 500px;
            height: 200px;
            border: 1px solid red;
            //父元素使用overflow
            overflow: auto;
        }

        .box span {
            //子元素使用flex:none避免掉子元素自动压缩
            flex: none;
            width: 100px;
            height: 50px;
            background-color: red;
            margin: 10px;
        }
    </style>

</head>

<body>
    <div class="box">
        <span>1</span>
        <span>2</span>
        <span>3</span>
        <span>4</span>
        <span>5</span>
        <span>6</span>
    </div>

</body>

</html>
